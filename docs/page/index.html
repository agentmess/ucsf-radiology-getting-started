Matlab Tool Resources

Posted on January 1, 0001

Matlab Resources The servers have many versions of Matlab. Using Matlab is quite simple, all it requires is typing the following command into a terminal in a VNC server: matlab & The default matlab is usually R2015b, although this could change. In order to use a specific version, specify which version you would like to run: e.g. matlab_2013a & To check all versions of matlab available in the server, run the following command in a terminal: bash -c "compgen -c | grep -E '^matlab.
